# Starting client back-end 
1. npm -i
2. npm start

http://localhost:8081

# Starting client front-end
1. npm -i
2. npm run serve

http://localhost:8080

# How to create the database
1. Run the `init.mysql` file
2. Run the `seed.mysql` file

For future runs, take note of the user privilege for the database and tables made.
